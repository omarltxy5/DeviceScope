## DeviceScope is a native, privacy-focused diagnostic and system inspection tool built on Jetpack Compose and Material 3. It translates complex system properties and `/proc` signals into a human-readable format to help power users, ROM maintainers, and developers understand their device environment.

DeviceScope is built w/ a focus of privacy and transperancy:
- **No Internet Permission:** The app cannot send data to a remote server.
- **No Analytics:** No Firebase, no telemetry, no tracking.
- **Local Data:** All information is read directly from your device and stays on your device.
- **FOSS:** Licensed under GPL v3, you can review, modify, and publish the code.

## Built With

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3)
- **Data Persistence:** [Jetpack DataStore](https://developer.android.com/topic/libraries/architecture/datastore)
- **Architecture:** 100% Native Android with **1** 3rd-party library [RootBeer](https://github.com/scottyab/rootbeer)
